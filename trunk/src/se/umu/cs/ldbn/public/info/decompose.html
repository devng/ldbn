Decompose in 2nd, 3rd NF or BCNF

The user must dive a set of relations, representing a valid decomposition of the initial relationship in 2nd, 3rd NF or BCNF. In every relation the user must specify the attributes, FDs and a key for the relation.

Add new Relation button

Shows a dialog window, where the user can input FDs. Furthermore, the attributes, contained in the FDs, will be automatically added in the attribute section of the relation.
