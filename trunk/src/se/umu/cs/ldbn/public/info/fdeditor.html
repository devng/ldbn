FD Editor

The FD editor window allow the users to define FDs. These are defined by giving the left and right hand side attributes of the FD.
Attributes muss be separated by commas, and they must be present at the “Given attributes” section, otherwise they wont be included in the FD.
The editor also supports Drag and Drop. User can drag any attributes or FDs and drop it in the text areas of the widget.

Set/Add button

Creates the FD from the attributes given in the two text boxes. Note that the dialog window wont close after pressing this button, so users can further define FD.

Clear button

Clears the text boxes from any given text inside.

Close button

Closes the dialog window.
