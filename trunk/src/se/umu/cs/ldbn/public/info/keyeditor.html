Key Editor

The Key editor window allow the users to define a key for a relation. This is defined by giving the attributes, of witch the key is build.

Attributes muss be separated by commas, and they must be present at the “Given attributes” section, otherwise they wont be included in the FD.

The editor also supports Drag and Drop. User can drag any attributes or FDs and drop it in the text areas of the widget.

Set button

Creates the key from the attributes given in the text box.

Clear button

Clears the text boxes from any given text inside.

Close button

Closes the dialog window.
