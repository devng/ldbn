Administrators

This tab is only visible for administrators and super users. Here they can manage the user rights of other users and/or delete assignments.

Add Administrators

This opens a dialog from which a user can be chosen and then added to the list of administrators. Please note that administrators can add regular users to the list of administrators but they cannot remove other administrators from that list, this can only be done by super users.

Remove Administrators

This button is only available for super users. It opens a dialog where super user can remove administrator rights of a user.

Delete Assignments

Deletes an assignment from the database and all comments associated with it.
