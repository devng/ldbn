Create assignment

This tab opens a widget, where registered users can create an assignment and save it in the database, or edit an existing assignment. An assignment consists of attributes and FDs, which are representing a relationship.

New button

Clears any input without waring.

Save button

Saves or updates an assignment in the DB. An assignment is updated only when it was previously loaded by using the Edit button. A label, displaying “Edit mode”, is shown, in order to notify the user, when an assignment is about to be updated, rather then saved as a new entity in the database.

Edit button

Loads an assignment from the DB. Next time you hit the Save button, the assignment will be updated in the DB, a label, displaying “Edit mode”, is indicating this.
