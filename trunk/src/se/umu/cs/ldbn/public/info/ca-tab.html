Create assignment

This tab opens a widget, where registered users can create an assignment and save it in the database, edit an existing assignment, or export it as an XML file to the local file system, or even import an assignment from an XML file.
An assignment consists of a relational-database schema in universal-relation form (URF), i.e., all the attributes in a single relation and a set of FDs on the attributes.

New button

Clears any input without waring.

Save button

Saves or updates an assignment in the DB. An assignment is updated only when it was previously loaded by using the Edit button. A label displaying "Edit mode" is shown in order to notify the user when an assignment is about to be updated, rather then saved as a new entity in the database.

Edit button

Loads an assignment from the DB. Next time you hit the Save button, the assignment will be updated in the DB. A label displaying "Edit mode" is indicating this.

Export button

Exports the current working assignment to the local file system, thus an assignemtn does not need to be stored in the DB in order to export it.

Import button

Imports an assignment from an XML file from the local file system. The imported assignment is set as a working assignment, thus it is not stored in the DB, in order to save it in the DB use the "Save" button.

Load in SA Tab button

Loads an assignment in the Solve Assignment tab without saving it in the database first. You do not need to log in to use this function. This functionality could be useful for testing purposes.
