Add/Remove Attributes

The widget allows users to add or remove attributes. Attributes are removed if they are not present in the text box, and attributes are added if they are present.

Attributes must be separated by commas or white space, and they must be present in the "Given attributes" section, otherwise they will not be included.

The editor supports also Drag and Drop. User can drag any attributes or FDs and drop them in the text area of the widget.

Set button

Sets the new attributes by parsing the the attributes given in the text box.

Clear button

Clears the text box from any given text inside.

Close button

Closes the dialog window.
