Add/Remove Attributes

The widget allows the user to add or remove attributes, by removing any attributes that are not present in the text box, and adding every attribute, which is present.

Attributes muss be separated by commas, and they must be present at the “Given attributes” section, otherwise they wont be included in the FD.

The editor also supports Drag and Drop. User can drag any attributes or FDs and drop it in the text areas of the widget.

Set button

Sets the new attributes by parsing the the attributes given in the text box.

Clear button

Clears the text box from any given text inside.

Close button

Closes the dialog window.
