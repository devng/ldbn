Solve assignment tab

This tab opens a widget, where students can give a solution to a given assignment. An assignment consists of attributes and FDs, which are representing a relationship. The user (the student) then have to find different decompositions of the relation.

New assignment button

Loads a new assignment from a data base. A dialog window will prompt the user to choose an assignment from a list of predefined assignments.

Check Solution button

Checks the solution of the user. A series of checks are being computed and the result is shown in a information window. The user are given small hints were the mistake might be.

Show Solution button

Shows a solution of the assignments. Please note there can be various solutions, but the program is showing only one of them. Furthermore the “checking solution” algorithms work separately from the “show solution” algorithms, thus checking the solution is not based on the solution shown by the program.
