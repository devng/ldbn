Key Editor

The Key editor window allows users to define a key for a relation. This is done by specifying the attributes, witch are building the key.

Attributes must be separated by commas or white space, and they must be present in the "Given attributes" section, otherwise they will not be included in the key.

The editor supports also Drag and Drop. User can drag any attributes or FDs and drop them in the text area of the widget.

Set button

Creates the key from the attributes given in the text box.

Clear button

Clears the text box from any given text inside.

Close button

Closes the dialog window.
